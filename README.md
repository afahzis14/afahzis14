### âš¡ï¸ Mulai Cepat


```bash
# 1) Clone repo
git clone https://github.com/afahzis14/github_profiling.git
cd github_profiling

# 2) Konfigurasi environment
cp .env.example .env
# Kemudian isi token GitHub Anda (PAT) di .env:
# GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxxx

# 3) Instal dependensi
# Pilih salah satu:
npm install
# atau
pnpm install
# atau
yarn

# 4) Jalankan pengembangan
npm run dev

# 5) Buka di browser
open http://localhost:3000
```
